# Kenya-Monetary-Policy-Stock-Market
Analyzing how interest rate changes affect NSE stock returns using econometrics and time-series models.
## Market
Nairobi Securities Exchange (NSE)

## Key Variable
Central Bank Rate (CBR)

## Research Question
Do interest rate changes affect stock returns on the NSE?

## Data Sources
- Central Bank of Kenya
- Nairobi Securities Exchange
- KNBS

## Methods
- Time Series Analysis
- Econometric Regression
- Volatility Modeling

## Project Structure

data/ - raw and processed datasets  
notebooks/ - research notebooks  
src/ - reusable code  
outputs/ - charts and tables
